# Top-Down Shooter Prototype (Unity 2D)

## Overview

A 2D top-down shooter prototype inspired by Box Head.
Built in Unity using pure 2D physics and systems-focused architecture.

## Core Features

- 8-directional player movement
- Mouse-based aiming
- Shooting mechanics
- Enemy waves & spawning
- Player health & damage
- Game over & restart system

## Tech Stack

- Unity (2D)
- C#
- Git & GitHub

## Controls

- Move: WASD
- Aim: Mouse
- Shoot: Left Mouse Button
